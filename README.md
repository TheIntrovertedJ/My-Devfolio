## About

This repository powers my personal developer portfolio. It’s configured to showcase my projects, blog posts, skills, and contact information in a way that reflects my experience and design preferences. The content, styles, and structure are tailored to my needs — update the source files to add new projects, tweak layouts, or change visual branding specific to my work.

## Features

- Simple, responsive layout
- Project showcase and filtering
- Blog support (optional)
- Easy customization with CSS/JS

## Getting Started

Prerequisites:

- Node.js (v22+)
- npm or yarn

- **Backend:** Node.js, Express, MongoDB

- **Source files:** `src/`

- **Serve production build:** `npm run start`

## My Journey

This portfolio was built to showcase my growth as a developer. I focused on creating a clean, performant experience that highlights my best work while maintaining a minimalist design aesthetic.

### Key Learnings

- Optimized performance through lazy loading and code splitting
- Implemented responsive design principles across all breakpoints
- Integrated modern tooling for streamlined development workflows
- Practiced accessibility standards (WCAG) for inclusive design
