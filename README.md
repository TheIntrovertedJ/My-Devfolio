## About

This repository powers my personal developer portfolio. It’s configured to showcase my projects, blog posts, skills, and contact information in a way that reflects my experience and design preferences. The content, styles, and structure are tailored to my needs — update the source files to add new projects, tweak layouts, or change visual branding specific to my work.

## Features

- Simple, responsive layout
- Project showcase and filtering
- Blog support (optional)
- Easy customization with CSS/JS

## Getting Started

Prerequisites:

- Node.js (v14+)
- npm or yarn

Quick start:

1. Clone the repo:
   git clone <repository-url>
   cd devfolio
2. Install dependencies:
   npm install
3. Update site content:
   - Replace placeholders (name, bio, contact links) in the relevant config files and src/ content.
   - Add projects to the projects data or markdown files as appropriate.
4. Run locally:
   npm run dev
5. Build for production:
   npm run build

## Development

- Source files: src/
- Build command: npm run build
- Serve production build: npm run start
